  • PETA SUNGAI

    This map aims to display the main river network, tributaries, and sub-watershed boundaries of the Tinalah River based on ALOS and TSAR imagery.   • PETA KEMIRINGAN LERENG

    This map aims to show the distribution of slope classes in the Tinalah sub-watershed, ranging from flat to steep, based on DEM, hydrography, and contour data. This information can be used for topographic analysis, land-use planning, and natural resource management in the area.

  • PETA ORDE SUNGAI

    This map aims to display the classification of river orders in the Tinalah watershed based on SRTM30 DEM data using the standard HydroTool method. This information helps analyze the hierarchy of the river network for water resource management planning and regional hydrological studies.

  • PETA TENTATIF HUJAN KRIGING

    This map aims to illustrate the distribution of regional rainfall intensity in the Bogowonto watershed using the Kriging method, thereby providing spatial information on rainfall variations in each area. The results can be used as a basis for planning water resource management, flood disaster mitigation, and more targeted land-use planning.

  • PETA INDEKS KEKERINGAN PERAIRAN

  • PETA INTERPOLASI AIR TANAH
